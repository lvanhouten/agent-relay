Badge — small uppercase mono label for shell names, PIDs, session counts, statuses.

```jsx
<Badge variant="accent">zsh</Badge>
<Badge variant="neutral">pid 48213</Badge>
<Badge variant="success">attached</Badge>
<Badge variant="warning">stale</Badge>
<Badge variant="outline">3 panes</Badge>
```

Variants: `neutral`, `accent`, `success`, `warning`, `danger`, `info`, `solid`, `outline`. Keep text to one or two short tokens.
